# Async Python local-bundle migration

The `/async-python` route is now generated by Astro from:

- `src/pages/async-python.astro`
- `src/components/AsyncPythonExplainer.tsx`

Before committing this update, remove the old legacy directory from the local repository:

```powershell
git rm -r public/async-python
```

Then copy this replacement project over the repository, stage the new files, commit, and push. The old public directory must not remain because its `index.html` would conflict with the new Astro route.
